# Avione — онлайн-платформа для бронирования авиабилетов

**Avione** — это веб-приложение для поиска, бронирования и покупки авиабилетов онлайн.  
Проект ориентирован на рынок Узбекистана и ближнего зарубежья, предоставляя пользователям удобный интерфейс, локальные валюты и актуальные предложения авиакомпаний.

---

## Описание проекта

Avione предназначен для упрощения процесса поиска и покупки авиабилетов.  
Сервис объединяет информацию от различных авиаперевозчиков и позволяет пользователю:

- находить рейсы по выгодным ценам;
- фильтровать результаты по дате, направлению и авиакомпании;
- бронировать и оплачивать билеты онлайн;
- получать актуальную информацию о рейсах и акциях.

Цель проекта — создать лёгкую, надёжную и визуально понятную систему бронирования, оптимизированную для локального рынка и пользователей всех уровней подготовки.

---

## Архитектура

Проект построен по принципам **Feature-Sliced Design (FSD)** и разделён на логические уровни для удобства поддержки и масштабирования.

avione_front/
├── app/ # Layout, глобальные стили, провайдеры
├── pages/ # Автоматические маршруты Nuxt
├── entities/ # Доменные сущности (tickets, users, cities, airlines и др.)
│ ├── tickets/
│ │ ├── api/
│ │ ├── model/
│ │ └── ui/
│ └── users/
│ ├── model/
│ └── ui/
├── features/ # Функциональные модули (фильтры, формы, шаги бронирования)
├── widgets/ # Крупные UI-блоки (поиск, результаты, футер)
├── shared/ # Общие библиотеки, UI-компоненты, типы
│ ├── ui/
│ ├── lib/
│ └── types/
├── composables/ # Nuxt-хуки (useAuth, useTheme и т.д.)
├── api/ # Клиент для REST-взаимодействия с backend
├── utils/ # Вспомогательные функции
├── public/ # Статические файлы (favicon, manifest)
└── nuxt.config.ts # Конфигурация Nuxt и SEO

## Технологический стек

- **Frontend:** Nuxt 3, Vue 3 (Composition API), TypeScript  
- **UI:** TailwindCSS  
- **State Management:** Pinia  
- **Архитектура:** Feature-Sliced Design  
- **API:** REST (Axios)  
- **DevOps:** Docker (в разработке)  
- **SEO:** ручная настройка meta-тегов  
- **i18n:** поддержка русского и английского языков

## Особенности реализации

- SSR и SEO-оптимизация.  
- Модульная архитектура с изоляцией бизнес-логики.  
- Адаптивный интерфейс для всех устройств.
